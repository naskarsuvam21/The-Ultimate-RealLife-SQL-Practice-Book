Here are all the **CSV files** provided. If you need to **import these CSV** files just like I did, you can watch a **tutorial on YouTube**, or I can explain the process again. In the **schema section**, click on the database you've prepared, and you will see the tables section. Right-click on the tables section, and you will find an option called **"Table Data Import Wizard"**. After clicking on that, it will ask for the **file path**. Provide the path where you downloaded the CSV files, select the separate CSV files one by one, and click open. Then, simply click next. After clicking next a few times, your file will be uploaded. 



এখানে প্রয়োজনীয় সব CSV ফাইল দেওয়া হলো। আপনি যদি আমার মতো করে এই CSV ফাইলগুলো ইম্পোর্ট করতে চান, তবে ইউটিউবে কোনো টিউটোরিয়াল দেখতে পারেন অথবা আমি প্রক্রিয়াটি আবারও বুঝিয়ে বলতে পারি। 'Schema' সেকশনে গিয়ে আপনার তৈরি করা ডেটাবেসটিতে ক্লিক করুন; এতে আপনি 'Tables' সেকশনটি দেখতে পাবেন। 'Tables' সেকশনের ওপর রাইট-ক্লিক করলে "Table Data Import Wizard" নামে একটি অপশন পাওয়া যাবে। সেখানে ক্লিক করার পর ফাইল পাথ (file path) বা ফাইলের অবস্থান জানতে চাওয়া হবে। যে ফোল্ডারে CSV ফাইলগুলো ডাউনলোড করেছেন সেই পাথটি দিন, এরপর একে একে আলাদা CSV ফাইলগুলো নির্বাচন করে 'Open'-এ ক্লিক করুন। এরপর শুধু 'Next'-এ ক্লিক করুন। কয়েকবার 'Next'-এ ক্লিক করলেই আপনার ফাইলটি আপলোড হয়ে যাবে।
